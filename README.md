# Click & Cook — Design & Documentation

## Описание

Данный репозиторий содержит **дизайн-документацию, архитектурные схемы и экономические модели** проекта **Click & Cook** — браузерной idle / clicker-игры с кулинарной тематикой.

Репозиторий используется как **единый источник проектных материалов**, на основе которых разрабатываются фронтенд и бэкенд игры.

> В репозитории **нет кода приложения**.  
> Здесь хранятся только документы, схемы, таблицы и проектные артефакты.

---

## Назначение репозитория

- Хранение **технического задания (ТЗ)**
- Описание **архитектуры приложения**
- Проектирование **базы данных**
- Проработка **игровой экономики и баланса**
- Документирование игровых механик
- Основа для синхронизации фронтенда и бэкенда

---

## Содержимое репозитория

### Техническая документация
- **`Веб-кликер ТЗ.pdf`**  
  Полное техническое задание на разработку игры:
  - жанр и концепция
  - игровые механики
  - этапы развития
  - престиж-система
  - требования к графике
  - технический стек

- **`Веб-кликер Архитектура.pdf`**  
  Описание архитектуры взаимодействия:
  - разделение логики фронтенд / бэкенд
  - REST API и WebSocket
  - жизненный цикл игровой сессии
  - защита от читерства

---

### Архитектура и база данных
- **`Архитектура БД.drawio.png`**  
  ER-диаграмма базы данных:
  - User / Session
  - Level / LevelXP
  - Prestige
  - Upgrade / Boost
  - SessionUpgrades

Используется как основа для реализации backend (PostgreSQL).

---

### Экономика и баланс
- **`Кликер Таблица улучшений.xlsx`**  
  Основной рабочий файл с экономикой игры:
  - блюда
  - оборудование
  - глобальные улучшения
  - персонал
  - рецепты
  - локации
  - коэффициенты роста
  - уровни доступа

- **`Кликер Таблица улучшений.pdf`**  
  Экспорт таблицы улучшений для просмотра и обсуждения.

Документы определяют:
- рост цен
- бонусы
- типы эффектов (mPc, dPs, mpM и т.д.)
- прогрессию по уровням и локациям

---

## Типы материалов

В репозитории используются следующие форматы:

- **PDF** — спецификации, архитектура, ТЗ
- **XLSX** — экономика и баланс
- **PNG / drawio** — схемы и диаграммы

---

## Связь с другими репозиториями

Данный репозиторий напрямую связан с:

- **clickNcook-front** — фронтенд (Angular)
- **clicker_api** — бэкенд (Go)

Все игровые механики, формулы и структура данных во фронте и бэке
должны основываться на документации из этого репозитория.

---

## Статус проекта

Репозиторий находится в **активном использовании**.  
Документы периодически обновляются по мере:
- изменения баланса
- переработки архитектуры
- добавления новых механик

---

## Лицензия

Все материалы предназначены для использования **в рамках проекта Click & Cook**.  
Коммерческое использование вне проекта — только с разрешения автора.
